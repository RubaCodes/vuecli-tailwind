# Hint Tailwind

Install the official tailwind intellisense add-on and inside class='' press ctr + space or add this to the 'config.json' file:

"tailwindCSS.includeLanguages": {
"html": "html",
"javascript": "javascript",
"css": "css"
},
"editor.quickSuggestions": {
"strings": true
}
